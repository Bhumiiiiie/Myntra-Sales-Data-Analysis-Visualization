# 🛍️ Retail Sales Data Analysis & Dashboard

## 📌 Project Overview

This project focuses on analyzing a retail dataset (including Clothing, 
Electronics, and Furniture categories) to extract meaningful business insights.
The workflow includes data cleaning, exploratory data analysis (EDA),
and building an interactive dashboard using Power BI.

The objective of this project is to transform raw, unstructured data into actionable insights that can help in decision-making.

---
## 🎯 Objectives

* Clean and preprocess raw dataset
* Perform Exploratory Data Analysis (EDA)
* Identify sales trends and patterns
* Build an interactive dashboard for visualization
* Generate business insights from data

---
## 📂 Dataset

* Dataset Name: Myntra Retail Dataset
* Contains information related to:

  * Product categories (Clothing, Electronics, Furniture)
  * Orders and sales
  * Dates and transactions

---
## 🧹 Data Cleaning (Python - Pandas)

Data cleaning was performed using Python in Jupyter Notebook.

### Key Steps:

* Removed missing/null values
* Eliminated duplicate records
* Standardized column names
* Converted data types (especially date columns)
* Handled inconsistent category labels
* Prepared dataset for analysis

---
## 📊 Exploratory Data Analysis (EDA)

EDA was conducted to understand patterns and trends in the dataset.

### Analysis Performed:

* Category-wise sales analysis
* Monthly sales trends
* Top-performing categories/products
* Revenue distribution

### Tools Used:

* Pandas
* Matplotlib
* Seaborn

---
## 📈 Dashboard (Power BI)

An interactive dashboard was created using Power BI to visualize key insights.

### Features:

* KPI Cards (Total Sales, Orders, etc.)
* Monthly Sales Trend Analysis
* Category-wise Sales Comparison
* Interactive filters (Category, Date)

---
## 🔍 Key Insights

* Clothing category contributes significantly to overall sales
* Certain months show higher sales indicating seasonal trends
* Electronics and Furniture categories show varying performance
* Sales distribution helps identify high-performing segments

---
## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Jupyter Notebook
* Power BI
* Data Visualization (Matplotlib, Seaborn)
